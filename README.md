## Installation

Execute:

    $ gem install gac

or

    $ bundle install

## Usage

To test the documentation generator, execute

    $ ./bin/gac-docgen input_file output_file(default: output.java)

To test the UI generator, do:

    $ ./bin/gac-uigen input_file output_file(default: output.cpp)
